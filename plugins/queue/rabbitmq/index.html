queue/rabbitmq plugin

This plugin delivers mails to rabbitmq queue for further processing.

Configuration

  • config/rabbitmq.ini This config file provides server address and port of rabbitmq server to deliver with other configs of queues and exchange.
    Example:
    [rabbitmq]; This is name of exchange.exchangeName  = emailMessages; ip and port of the server.server_ip = localhostserver_port = 5672; user and passworduser = guestpassword = guest; name of the queue which reader will readqueueName = email; This is for making it persistant while publishing messagedeliveryMode = 2; If true it will require ack for marking it complete from workerconfirm = true; Again for persistance passed while creating queuedurable = true; if true will delete queue if publisher quitsautoDelete = false; type of the exchangeexchangeType = direct

More information about rabbitmq can be found at https://www.rabbitmq.com/
